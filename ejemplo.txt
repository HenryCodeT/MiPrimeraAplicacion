Comandos basicos de git
ls- meustra la carpeta de archivos en la teminal
pwd- te indica donde estas ubicado actualmente 
cd NombreDeLaCarpeta-navegar interanmente en una carpeta
cd.. -Salir fuera de al carpeta actual
mkdir NombreDeLaCarpeta - Crear una carpeta nueva
touch NombreArchivo.extencion - crear un archivo nuevo
rm NombreArchivo.extencion - Eliminar archivo o carpeta
rm -d nombreDirectorio - Para eliminar directorio

Comandos de Git
1) git init - Inicializa una carpeta como un repositorio localmente
2) git add NombreArchivo git add .
3) git commit -m "Mensaje para el commit"
4) git remote add origin url-del-github - Enlaza carpeta a github
5) git push -u origin master git push -u origin main
6) git status - estado de los archivos
rm -rf .git
